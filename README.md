# SEO Checker

SEO Checker is a simple Next.js application that allows you to paste a URL and see basic SEO and analytics-related insights for that page.

Live app: https://seo-checker-mocha.vercel.app

## Features

* Paste any URL and run an SEO check
* View basic analytics and SEO-related results
* Fast UI built with Next.js
* Suitable for quick audits and experiments

## Tech Stack

* **Next.js** (App Router)
* **React**
* **TailwindCSS**
* **TypeScript**

## Getting Started

### Prerequisites

Make sure you have the following installed:

* Node.js (>= 18 recommended)
* npm or yarn or pnpm

### Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/JKob207/seo-checker
cd seo-checker
npm install
```

or with yarn:

```bash
yarn install
```

or with pnpm:

```bash
pnpm install
```

### Running Locally

Start the development server:

```bash
npm run dev
```

Then open your browser and go to:

```
http://localhost:3000
```

The app will reload automatically when you make changes.

### Production Build

To build the app for production:

```bash
npm run build
npm run start
```

## Usage

1. Open the app in your browser
2. Paste a URL into the input field
3. Submit
4. Review the SEO and analytics results displayed on the page
